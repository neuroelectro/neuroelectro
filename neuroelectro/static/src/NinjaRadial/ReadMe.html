How to start?

1. Ensure that jQuery is included in the head of the document (v1.9.3 or later).

<script src="http://code.jquery.com/jquery-1.11.3.min.js" type="text/javascript"></script>

2. Include NinjaRadial.css and NinjaRadial.js in the head of the document.

<link href="NinjaRadial.css" rel="stylesheet" type="text/css" />

<script src="NinjaRadial.js" type="text/javascript"></script>

3. Configure your buttons creating an object array:

//-- This is the basic usage

buttonsModel = new Array();

buttonsModel.push({ text: "Edit", click: function (e, obj) { alert("Edit: " + obj.id); } });

buttonsModel.push({ text: "Remove", click: function (e, obj) { alert("Remove: " + obj.id); } });

buttonsModel.push({ text: "Add", click: function (e, obj) { alert("Add: " + obj.id); }});

5. And then, call ninjaRadial puglin with the desired options.

$("#ninjaRadial").ninjaGridPlus({

buttons: buttonsModel

});

6. Enjoy it!

Buttons Options

Button's properties options

{

//-- this is the button label

text: 'Edit'

 

//-- optionally, set a custom css with FontAwesome, or you can use your own icon, image...

cssIcon: 'fa fa-pencil'

 

//-- optionally, you can set a css class to the label button too...

cssLabel: 'editLabel'

 

//-- set a function callback to the user click, each button can call a diferent function...

click: 'function (e, obj) { alert("Edit: " + obj.description); }'

}

NinjaRadial Options

NinjaRadial properties options

{

//-- this is background container size in pixels

buttons: buttonsModel

//-- you can change the size of the background container in pixels

innerSize: 100

 

//-- this is background container border size in pixels

borderSize: 10

 

//-- set the distance between center and buttons in pixels

distance: 60

 

//-- change the background container color

backColor: 'rgba(20,20,20,0.7)'

 

//-- change the background container border color

borderColor: 'rgba(20,20,20,0.1)'

 

//-- you can set the button's height in pixels

itemHeight: 40

 

//-- you can set the button's width in pixels

itemWidth: 40

 

//-- you can set the line's height in pixels

lineHeight: 40

 

//-- use this function to conditionally create or change buttons

onLoad: function (element, options)

{

//-- do something in here

}

}
